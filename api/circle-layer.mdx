---
title: CircleLayer
description: Renders filled circles on the map
---

CircleLayer is a style layer that renders one or more filled circles on the map.

[Mapbox Style Specification](https://docs.mapbox.com/mapbox-gl-js/style-spec/layers/#circle)

## Props

<ParamField path="id" type="string" required>
  A unique identifier for the layer.
</ParamField>

<ParamField path="existing" type="boolean">
  If true, references an existing layer in the style instead of creating a new one.
</ParamField>

<ParamField path="sourceID" type="string" default="Mapbox.StyleSource.DefaultSourceID">
  The source from which to obtain the data to style. Inferred from parent source only if the layer is a direct child to it.
</ParamField>

<ParamField path="sourceLayerID" type="string">
  Identifier of the layer within the source identified by sourceID.
</ParamField>

<ParamField path="aboveLayerID" type="string">
  Inserts the layer above the specified layer ID.
</ParamField>

<ParamField path="belowLayerID" type="string">
  Inserts the layer below the specified layer ID.
</ParamField>

<ParamField path="layerIndex" type="number">
  Inserts the layer at a specified index in the layer stack.
</ParamField>

<ParamField path="slot" type="'bottom' | 'middle' | 'top'">
  The slot this layer is assigned to. If specified, and a slot with that name exists, it will be placed at that position in the layer order.
  
  v11 only
</ParamField>

<ParamField path="minZoomLevel" type="number">
  The minimum zoom level at which the layer is visible.
</ParamField>

<ParamField path="maxZoomLevel" type="number">
  The maximum zoom level at which the layer is visible.
</ParamField>

<ParamField path="filter" type="FilterExpression">
  Filters features from the source layer based on a condition.
</ParamField>

<ParamField path="style" type="CircleLayerStyleProps">
  Customizable style attributes for the circle layer.
</ParamField>

## Style Properties

<ParamField path="style.visibility" type="'visible' | 'none'" default="visible">
  Controls whether the layer is displayed.
</ParamField>

<ParamField path="style.circleRadius" type="number" default="5">
  Circle radius in pixels. Minimum: 0
</ParamField>

<ParamField path="style.circleColor" type="string" default="#000000">
  The fill color of the circle.
</ParamField>

<ParamField path="style.circleOpacity" type="number" default="1">
  The opacity at which the circle will be drawn. Range: 0-1.
</ParamField>

<ParamField path="style.circleStrokeWidth" type="number" default="0">
  The width of the circle's stroke in pixels. Strokes are placed outside of the circleRadius. Minimum: 0
</ParamField>

<ParamField path="style.circleStrokeColor" type="string" default="#000000">
  The stroke color of the circle.
</ParamField>

<ParamField path="style.circleStrokeOpacity" type="number" default="1">
  The opacity of the circle's stroke. Range: 0-1.
</ParamField>

<ParamField path="style.circleBlur" type="number" default="0">
  Amount to blur the circle. 1 blurs the circle such that only the centerpoint is full opacity.
</ParamField>

## Example

```tsx
import { MapView, ShapeSource, CircleLayer } from '@rnmapbox/maps';

const points = {
  type: 'FeatureCollection',
  features: [
    {
      type: 'Feature',
      geometry: {
        type: 'Point',
        coordinates: [-122.4, 37.8],
      },
      properties: { name: 'San Francisco' },
    },
  ],
};

function Map() {
  return (
    <MapView>
      <ShapeSource id="points" shape={points}>
        <CircleLayer
          id="points-circle"
          style={{
            circleRadius: 8,
            circleColor: '#3887be',
            circleOpacity: 0.8,
            circleStrokeWidth: 2,
            circleStrokeColor: '#ffffff',
          }}
        />
      </ShapeSource>
    </MapView>
  );
}
```
